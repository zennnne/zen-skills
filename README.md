# zen-skills

Zen's personal Claude Code skill collection.

## Skills

| Skill | Description |
|-------|-------------|
| `applying-3-act-structure` | Structure content using 3-act narrative framework |
| `audit-skills` | Scan and improve skill descriptions for discoverability |
| `auditing-security` | Security audit for code changes |
| `cleaning-sessions` | Clean up session logs |
| `craft-skill` | Research a topic and craft a new SKILL.md |
| `crafting-skills-deeply` | Deep skill authoring with research and critique |
| `designing-agent-roles` | Design subagent roles, tool scope, and system prompts |
| `designing-subagent-systems` | Design multi-agent orchestration systems |
| `extracting-youtube-transcript` | Extract and process YouTube transcripts |
| `harvest-insights` | Extract key insights from content |
| `reviewing-code` | Code review for correctness and quality |
| `session-index` | Generate monthly session index |
| `session-summary` | Backfill daily session summaries |
| `updating-plugins` | Update Claude Code plugins |
| `setup-session-log` | Install Zen's auto session logging system on a new machine |

## Install

> **Note:** This is a private repo — ต้อง `gh auth login` ก่อนนะ

**1. Add marketplace**
```
/plugin marketplace add zennnne/zen-skills
```

**2. Install plugin**
```
/plugin install zen-skills@zen-skills
```
